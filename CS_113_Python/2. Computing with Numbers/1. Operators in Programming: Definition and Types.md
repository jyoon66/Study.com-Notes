Learn how operators in programming work. Understand how they enable calculations, comparisons, and logical control by working with operands in expressions.

### Introduction to Operators in Programming
All programming languages depend on **operators**, as they serve as tools to perform various operations on data. Operators are the basic elements that allow programs to execute expressions, perform calculations and comparisons, and make logical decisions.

### Operators in Programming Logic
Programs use operators to create expressions that determine program decisions and data manipulation operations. This section explains operator functionality and shows their application for constructing operations in code.

### How Operators Work in Code
The operators perform operations on one or more operands which include variables or constants. For example, in the expression a + b, + is the operator, and a and b are operands. The operation result depends on the operator's function: arithmetic operators return numeric results, and relational operators return Boolean values.

Example of Operands and Operator:

a = 5

b = 3

result = a + b

print(result) #Output: 8

### Operators and Expressions
Operators are essential for forming **expressions**. These are combinations of variables, values, and operators that Python evaluates to produce another value. Most programming work consists of expressions ranging from basic arithmetic operations to sophisticated logical tests.

Example of an expression:

x = 10

y = 4

expression = (x * 2) + (y - 1)

print(expression) #Output: 23

### Arithmetic Operators: Performing Mathematical Calculations
Programs use arithmetic operators to execute mathematical operations. These operators operate on numeric data types and represent the fundamental operators found in every programming language.

### Types of Arithmetic Operators
The arithmetic operators perform basic math operations such as addition, subtraction, multiplication, and division. Python has various arithmetic operators used in numeric data types such as integers and floats.

* + Addition
* - Subtraction
* (*) Multiplication
* / Division
* // Floor Division
* % Modulus
* (**) Exponentiation

Python arithmetic operator example:

a = 9

b = 2

print(a + b) #Output: 11

print(a - b) #Output: 7

print(a * b) # Output: 18

print(a / b) # Output: 4.5

print(a // b) # Output: 4

print(a % b) # Output: 1

print(a ** b) # Output: 81

### Relational Operators: Comparing Data
Relational operators are used when the program needs to compare the values and do something based on the result. Relational operators are used in the conditional logic, such as whether a number meets some condition or not.

### Purpose of Relational Operators
Relational operators or comparison operators are used to compare two values. They return Boolean results: either True or False. These operators are a part of the decision-making processes, especially in conditional statements like if, elif, and while.

* == Equal to
* != Not equal to
* > Greater than
* < Less than
* >= Greater than or equal to
* <= Less than or equal to

Python relational operator example:

x = 7

y = 10

print(x == y) # Output: False

print(x != y) # Output: True

print(x > y) # Output: False

print(x < y) # Output: True

print(x >= 7) # Output: True

print(y <= 10) # Output: True

### Logical Operators: Combining Conditions
Programmers use logical operators to merge multiple comparison statements or Boolean values into a single expression. They enable programs to make decisions through multiple conditions simultaneously.

### Using Logical Operators
Logical operators take two or more conditions or Boolean values and combine them. They are mostly used where more than one condition has to be checked using control flow statements. Logical operators result in True or False depending on the combined value.

* and: True if both expressions are True
* or: True if at least one expression is True
* not: Inverts the Boolean value

Python logical operator example:

a = 5

b = 8

print(a > 0 and b < 10) # Output: True

print(a < 0 or b < 10) # Output: True

print(not (a > 0)) # Output: False

### Assignment Operators: Storing Values in Variables

Assignment operators are used to store values in variables while simultaneously executing operations that update these values. These operators are essential tools for controlling data throughout a program.

* = Assign
* += Add and assign
* -= Subtract and assign
* (*=) Multiply and assign
* /= Divide and assign
* %= Modulus and assign
* (**)= Exponentiate and assign

Python assignment operator example:

x = 10

x += 5 # x = x + 5

print(x) # Output: 15

x *= 2 # x = x * 2

print(x) # Output: 30

x **= 2 # x = x ** 2

print(x) # Output: 900

### Bitwise Operators: What Are Bitwise Operators?

The bitwise operators operate on individual bits of information. The bitwise operators provide programmers with a useful tool for performing low-level data processing tasks in systems programming or embedded systems. Python enables bitwise operations through the operators & (AND), | (OR), ^ (XOR), (NOT), << (left shift) and >> (right shift). For example, 5 & 3 performs a bitwise AND operation between the binary representations of 5 (0101) and 3 (0011). This results in 0001, which is 1 in decimal. Data manipulation at the bit level with bitwise operations can be useful in doing things like flags, masks, or performance-critical arithmetic.

* & Bitwise AND
* | Bitwise OR
* ^ Bitwise XOR
* Bitwise NOT
* << Left Shift
* >> Right Shift

Python bitwise operator example:

a = 5 # 0101

b = 3 # 0011

print(a & b) # Output: 1 (0001)

print(a | b) # Output: 7 (0111)

print(a ^ b) # Output: 6 (0110)

print(~a) # Output: -6

print(a << 1) # Output: 10

print(a >> 1) # Output: 2

### Identity and Membership Operators in Python

Python provides special operators for membership testing and object identity. This can be used to determine relationships between variables or if an item belongs to a collection.

### Identity Operators: Checking Object Identity
Identity operators check if two variables refer to the same object in memory, not just if their values are equal.

is
is not
Python identity operator example:

x = [1, 2, 3]

y = x

z = [1, 2, 3]

print(x is y) # Output: True

print(x is not z) # Output: True

### Membership Operators: Checking Inclusion
Membership operators test whether a value exists in a sequence such as a list, string, or set.

* in
* not in
Python membership operator example:

colors = ['red', 'green', 'blue']

print('red' in colors) # Output: True

print('yellow' not in colors) # Output: True

### Lesson Summary

Python **operators** enable developers to modify data while controlling program logic execution. An operator is a symbol that applies a specific action to one or more **operands**, which are values or variables that operators act upon. **Expressions**, which are combinations of variables, constants, and operators, are the basis of all programming logic. The arithmetic operators perform mathematical operations ranging from addition through exponentiation. Relational operators evaluate values to produce Boolean results, while logical operators enable users to merge Boolean expressions for advanced evaluations. The assignment operators provide compact syntax to change variable values, and bitwise operators enable binary operations for optimization attempts. Python provides identity operators to check variable object equality and membership operators to verify value presence in collections. The operator categories allow developers to build exact and readable code that executes numerous operations.
