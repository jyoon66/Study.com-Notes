## Quiz
1. What does the acronym CSV stand for in its full form?
   * Comma Separated Value
2. CSV files use delimiters to separate which data?
   * Column data
3. CSV files contain which type of data?
   * Tabular data
4. What does the first row in the CSV file typically contain?
   * The column names of the data
5. What is the default delimiter of a CSV file
   * Comma
