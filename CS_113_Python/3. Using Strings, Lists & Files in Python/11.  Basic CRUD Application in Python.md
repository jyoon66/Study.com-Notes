## Quiz
1. What is the purpose of the commit() method in a Python MySQL database operation?
   * It saves changes made by SQL queries.
2. A developer wants to update a worker's salary in the tblemployee table. Which SQL command should they use in Python?
   * UPDATE tblemployee SET salary=60000 WHERE emp_id=2
3. Which SQL query is used in Python to retrieve all records from the tblemployee table?
   * SELECT * FROM tblemployee
4. A developer uses Python and MySQL to delete an employee's record with emp_id = 3. Which code achieves this?
   * DELETE FROM tblemployee WHERE emp_id = 3
5. What does the CRUD acronym represent in software development?
   * Create, Read, Update, Delete
