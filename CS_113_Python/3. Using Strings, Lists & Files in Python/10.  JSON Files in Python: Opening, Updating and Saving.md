## Quiz
1. What parameter can be passed to json.dump() to improve human readability of the saved file?
   * indent=4
2. Which two Python data types most commonly correspond to JSON arrays and JSON object?
   * Lists and dictionaries
3. An app attempts to load configuration settings from a file named settings.json, but the file contains a syntax error. Which error will be raised, and how should it be handled?
   * json.JSONDecodeError, to be caught using a try...except block
4. What is the main purpose of the json.load() function in Python?
   * To read JSON data from a file and convert it to a Python object
5. A developer wants to change a value in inventory.json and save the update. What is the correct approach?
   * Read, update, and then write the JSON using json functions
