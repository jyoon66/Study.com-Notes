## Quiz

1. Which re function is used to ensure that the entire string, and not just a part, matches a given pattern?
   * re.fullmatch()
2. A user inputs a string containing characters like +, (, and ?. To safely search for that exact string in another sentence, which re function should be used?
   * re.escape()
3. Which module must be imported to use regular expression features in Python?
   * re
4. A Python script converts "12/25/2024" into "2024-12-25" using the re.sub() function. Which part of the regular expression enables this transformation?
   * Grouped digits using parentheses like r" (\d{2})/(\d{2})/(\d{4})
5. What does the pattern r"\(\d{3}\) \d{3}-\d{4}" validate in a string?
   * A US phone number
