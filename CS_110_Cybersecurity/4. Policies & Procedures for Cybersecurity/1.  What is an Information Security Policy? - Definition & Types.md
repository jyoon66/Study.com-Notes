### Lesson Summary
**Information security policies** protect information from unauthorized access. They can occur at the **regulatory**, **administrative** and **technological** levels, and these levels can all interact.


### Quiz
1. What is one of the major challenges of information security policies?
   * The administrative, regulatory and technological policies can all interact.
2. What is an example of an administrative information security policy?
   * Making a policy to shred confidential documents when they're disposed of.
3. HIPAA is an example of a(n) _____ information security policy?
   * Regulatory.
4. A(n) _____ defines the risk of information compromising security if it's leaked.
   * Risk assessment.
5. Requiring users to change their passwords is an example of a(n) _____ security policy?
   * Technological.
